# eventos
CRUD com Python + Flask

Criar um ambiente virtual
  > python3 -m venv venv

Ativar o ambiente virtual
  No Linux:
  > . venv/bin/activate
  No Windows:
  > venv\Scripts\activate

Instalar o Flask
  > pip install flask

Executar o Flask
  > flask run

Se precisar instalar o conector MySQL
  > pip install mysql-connector-python
